Detailed Description

C++ wrapper around librdf_statement using RAII for memory management.

construct a LibrdfStatement from LibrdfNode objects.

The memory associated with the constructed librdf_statement* is managed by RAII while the LibrdfNode types handled themselves, also by RAII. The reference count of the LibrdfNode types is incremented by 1 on instantiation
