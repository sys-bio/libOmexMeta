Stripping Annotations¶

Here are some examples of how to remove annotations from an xml document

Remove annotation elements from an sbml document in Python¶
 1from pyomexmeta import RDF, Editor
 2
 3sbml = """<?xml version="1.0" encoding="UTF-8"?>
 4<sbml xmlns="http://www.sbml.org/sbml/level3/version1/core" level="3" version="1">

 6        <listOfCompartments>
 7            <compartment id="cytosol" metaid="comp1" spatialDimensions="3" size="1" constant="true"/>
 8            <compartment id="extraCell" metaid="comp2" spatialDimensions="3" size="10" constant="true"/>
 9        </listOfCompartments>
10        <listOfSpecies>
11            <species id="A" metaid="sp_1" compartment="cytosol" initialConcentration="10" hasOnlySubstanceUnits="false"
12                     boundaryCondition="false" constant="false">

14                    <rdf:RDF xmlns:rdf="http://www.w3.org/1999/02/22-rdf-syntax-ns#"
15                             xmlns:bqbiol="http://biomodels.net/biology-qualifiers/">
16                        <rdf:Description rdf:about="#DescriptionAnnotationWithBag">
17                            <bqbiol:is>
18                                <rdf:Bag>
19                                    <rdf:li rdf:resource="http://identifiers.org/uniprot/P00742"/>
20                                    <rdf:li rdf:resource="http://identifiers.org/uniprot/P00743"/>
21                                </rdf:Bag>

23                        </rdf:Description>

64                    <speciesReference species="PlasmaCa" stoichiometry="1" constant="true"/>
